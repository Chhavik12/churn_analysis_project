# 🚀 Customer Churn Prediction & Analysis Dashboard

## 📌 Project Overview

This project builds an end-to-end Machine Learning system to analyze customer behavior and predict churn using a telecom dataset.

It combines data preprocessing, machine learning, and Power BI visualization to simulate real-world business decision-making.

---

## 🎯 Objectives

* Analyze customer churn patterns
* Identify key factors influencing churn
* Build predictive models for churn classification
* Segment customers based on churn risk
* Provide actionable business insights via dashboard

---

## 🧠 Features

* ✅ Data preprocessing and feature engineering
* ✅ Churn prediction using Machine Learning models
* ✅ Model comparison (Logistic Regression & Random Forest)
* ✅ Feature importance analysis
* ✅ Risk segmentation (High / Medium / Low)
* ✅ Interactive Power BI dashboard

---

## 🛠️ Tech Stack

* **Python** (Pandas, NumPy, Scikit-learn)
* **Machine Learning** (Logistic Regression, Random Forest)
* **Power BI** (Dashboard Visualization)
* **Excel** (Data source)

---

## 📊 Dashboard Overview

### 🔹 Page 1: Executive Overview
Provides a high-level summary of business KPIs:
- Total Customers
- Churn Rate
- Total Revenue
- Monthly Recurring Revenue (MRR)

---

### 🔹 Page 2: Customer Churn Drivers Analysis
Analyzes factors contributing to churn:
- Churn by Contract Type
- Churn by Tenure
- Churn by Monthly Charges
- Churn by Payment Method

---

### 🔹 Page 3: Revenue Insights
Examines revenue contribution across segments:
- Revenue by Subscription Plan
- Revenue by Customer Tenure
- Revenue by Monthly Charges

---

### 🔹 Page 4: Churn Prediction
Displays model predictions and risk segmentation:
- High-risk customers
- Medium-risk customers
- Churn probability distribution
- Customer-level prediction table

## 🚀 How to Run the Project

1. Clone the repository  
2. Open the notebook (`churn_analysis.ipynb`)  
3. Run data preprocessing and model training  
4. Generate predictions  
5. Load processed data into Power BI dashboard  

---

## 🎯 Key Insights

* Customers with month-to-month contracts show highest churn  
* Low tenure customers are more likely to leave  
* Higher monthly charges increase churn probability  
* High-value customers also show significant churn risk
